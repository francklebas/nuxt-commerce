# Branding and Design Guide

## Brand direction

- Positioning: Quiet luxury, minimal tailoring, modern capsule
- Tone: Calm, clear, premium, never overhyped

## Visual system

- Keep spacing generous
- Prefer subtle contrast over loud colors
- Use image-driven storytelling with clean typography

## Typography

- Display: Serif for headings (editorial tone)
- Body: Sans-serif for readability and UI clarity

## Product copy rules

- First line: material + silhouette
- Second line: practical fit details
- Third line: emotional value statement

## Interaction rules

- Hover on cards: subtle scale only
- Buttons: smooth transition, soft radius
- Dark mode: maintain strong text contrast at all times
